发表于 2020-10-23

下面就来分享一下本博客的搭建流程

前期准备

1.安装 Node.js 2.安装 Git 3.注册 Gitbub 账号

服务器端

要想有自己的博客网站，搭建自己的服务器是必不可少的。常见的方式有两种： 1.够买云服务器，自己搭建环境 2.使用第三方服务提供的环境，如 Github Pages
本站采用的是第二种，在这里详细说明一下

1.登录 Github 账号，新建一个仓库

2.设定仓库的 Github Pages 配置

3.点击该链接即可浏览你的博客网站啦~这里默认使用自带的主题模板，可以在 setting 里更换主题
当然也可以自己编写 index.html 放在该目录下

安装 Hexo

网站已初见眉目，剩下就是利用 Hexo 去打造自己喜爱的博客系统和文章啦

什么是 hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页

安装 hexo

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

安装完后输入 hexo -v 验证是否安装成功。
